Load Existing Server

Use this page to import an existing Minecraft server directory into MC-Manager.

Your server folder must contain:

  • docker-compose.yml — used to read the server version, type, and port
  • data/ — your world, player data, and server.properties (data/ is optional; new server and world will be created)

What happens when you load:

  1. The entire folder is copied into MC-Manager's server storage and your original files are left untouched.
  2. The version, mod loader, and RAM limit are read from docker-compose.yml automatically.
  3. If the original port is already in use, a new port is assigned.
  4. The server appears on your Dashboard, ready to start.
